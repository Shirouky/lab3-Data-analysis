# Лабораторная работа №3: Обработка и анализ данных

## Описание
Обработка данных из JSON-файла с товарами бисера: очистка, заполнение пропусков, фильтрация и анализ с использованием Pandas.

## Технологии
- **Python 3.7+**
- **Pandas** - обработка данных
- **NumPy** - числовые операции
- **OpenPyXL** - работа с Excel

## Структура проекта
```
LW3.ipynb                 # основной скрипт обработки
beads.json                # исходные данные, полученные из ЛР2
DataFrame beads.xlsx      # результат (генерируется)
```

## Функциональность

### Обработка пропущенных значений
- **Случайное создание пропусков** в категориальных данных (цвет)
- **Заполнение категориальных пропусков** модальным значением
- **Заполнение числовых пропусков** методом "fill forward"

### Очистка данных
- **Удаление дубликатов** и некорректных записей
- **Фильтрация абсурдных значений** (цена < 10, пустые цвета)
- **Нормализация кодов товаров** - удаление пробелов и форматирование

### Анализ данных
- **Факторизация категориальных признаков** (цвет, тип, код)
- **Сортировка по значимым параметрам**
- **Фильтрация по среднему значению** цены с использованием масок
- **Конкатенация параметров** в единую строку

### Экспорт результатов
- **Сохранение в Excel** с сохранением структуры данных
